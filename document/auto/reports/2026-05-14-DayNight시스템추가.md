# DayNight 시스템 추가

WES에 낮/밤 사이클을 도입하고, 밤 전용 콘텐츠(시야 제한, 야간 몬스터)와 야간 렌더 효과를 추가했다.

## 시퀀스

```mermaid
sequenceDiagram
    participant Server as Server (DayNightWorker)
    participant NetVar as NetworkVariable<int> m_Phase
    participant Client as Client
    participant Listeners as 구독자 (NightVision/Monster/Render)

    Server->>NetVar: 페이즈 갱신 (Day→Night)
    NetVar-->>Client: OnValueChanged 동기화
    Client->>Listeners: OnPhaseChanged(oldPhase, newPhase) 발사
    Listeners->>Listeners: 야간 시야/스폰/렌더 토글
```
