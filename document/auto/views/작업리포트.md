# 작업 리포트

에이전트가 작업 완료 시 자동 생성하는 리포트 모음.

## 최근 작업 (날짜 정렬)

```dataview
TABLE
  date AS "날짜",
  area AS "영역",
  status AS "상태",
  team AS "팀",
  affected AS "영향 클래스"
FROM "auto/reports"
SORT date DESC
LIMIT 30
```

## 상태별 카운트

```dataview
TABLE WITHOUT ID status AS "상태", length(rows) AS "개수"
FROM "auto/reports"
GROUP BY status
SORT length(rows) DESC
```

## 진행 중 (In Progress / Blocked)

```dataview
TABLE date AS "날짜", area AS "영역", status AS "상태"
FROM "auto/reports"
WHERE status = "In Progress" OR status = "Blocked"
SORT date DESC
```

## Discord 미노출 (vault 전용)

```dataview
TABLE date AS "날짜", team AS "팀"
FROM "auto/reports"
WHERE discord_notified = false
SORT date DESC
```
