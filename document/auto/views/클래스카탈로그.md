# 클래스 카탈로그

자동 생성된 클래스 목록. Dataview 플러그인이 `auto/catalog/Class/` 하위 .md 파일들의 frontmatter를 읽어 표로 렌더링한다.

## 전체 목록

```dataview
TABLE
  category AS "카테고리",
  parent AS "부모",
  role AS "역할",
  status AS "상태"
FROM "auto/catalog/Class"
SORT category ASC, name ASC
```

## Controller만

```dataview
TABLE
  parent AS "부모",
  role AS "역할",
  status AS "상태"
FROM "auto/catalog/Class"
WHERE category = "Controller"
SORT name ASC
```

## 카테고리별 카운트

```dataview
TABLE WITHOUT ID category AS "카테고리", length(rows) AS "개수"
FROM "auto/catalog/Class"
GROUP BY category
SORT length(rows) DESC
```

---

## 검증 가이드

이 페이지를 Obsidian에서 열었을 때:

- ✅ 위 코드블록 3개가 **표 형태로 렌더링**되면 Dataview 정상 작동
- ✅ 첫 표에 `InGameController`, `GameController` 2개 row가 보여야 함
- ❌ 코드블록 텍스트 그대로 보이면 Dataview 플러그인 미설치/미활성화 상태
