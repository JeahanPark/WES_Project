# 시그널·이벤트 카탈로그

WES의 이벤트/시그널/RPC/NetworkVariable 등 코드 간 통신 채널을 모은다.

## 전체 목록

```dataview
TABLE
  kind AS "종류",
  owner AS "발사 주체",
  signature AS "시그니처",
  frequency AS "주기",
  status AS "상태"
FROM "auto/catalog/Signal"
SORT kind ASC, name ASC
```

## 종류별 카운트

```dataview
TABLE WITHOUT ID kind AS "종류", length(rows) AS "개수"
FROM "auto/catalog/Signal"
GROUP BY kind
SORT length(rows) DESC
```

## Active만

```dataview
LIST owner
FROM "auto/catalog/Signal"
WHERE status = "Active"
SORT name ASC
```
