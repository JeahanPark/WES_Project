# 맵 설계 — M2 Week 2

---

## 컨셉

- **섬 지형**: 난파 → 해안가 → 숲 → 산지 → 탈출 지점
- **진행 방향**: 맵 한쪽에서 반대쪽으로 이동하며 난이도 상승
- **에셋**: Synty PolygonGeneric (지형, 자연물, 환경)

---

## 맵 레이아웃

```
[탈출 지점] ← 산지(골렘) ← 숲(늑대) ← 해안가(Test01Monster) [스폰 지점]

넓이: 약 200m x 200m
```

### 영역별 구성

| 영역 | 위치 | 지형 | 몬스터 | 월드 오브젝트 | 분위기 |
|------|------|------|--------|-------------|--------|
| **해안가** | 남쪽 (스폰) | 평탄, 모래 | Test01Monster x3 | 참나무, 바위 | 안전, 자원 수집 |
| **숲** | 중앙 | 나무 밀집, 언덕 | 늑대 x4 | 소나무, 약초 덤불 | 긴장감 시작 |
| **산지** | 북쪽 (탈출) | 절벽, 바위 | 골렘 x2 | 바위, 철광석 바위 | 위험, 고급 자원 |

### 주요 배치물

| 배치물 | 에셋 (PolygonGeneric) | 영역 |
|--------|---------------------|------|
| 지면 타일 | Ground_Grass, Ground_Dirt | 전체 |
| 나무 | Tree_01~03, Tree_Pine_01~03 | 해안가, 숲 |
| 바위 | Rock_01~10 | 전체 |
| 덤불 | Bush_01~04 | 숲 |
| 풀 | Grass_01~07 | 전체 |
| 절벽 | Cliff_01~08 | 산지 |
| 산 | Mountain_01~03 | 산지 배경 |
| 물 | Water_Plane | 해안가 주변 |
| 꽃 | Flower_01~08 | 숲 |

---

## NavMesh

- 전체 맵에 NavMesh 굽기
- 절벽/물은 NavMesh 제외 (이동 불가 영역)
- 몬스터는 NavMeshAgent로 영역 내 배회

---

## 플레이어 경로

```
1. 스폰 (해안가 남쪽)
2. 주변 채집 (나무, 돌) → 기본 장비/소비품 제작
3. 숲 진입 → 늑대 처치, 가죽/뼈 획득 → 장비 업그레이드
4. 산지 진입 → 골렘 처치, 철광석 획득 → 철검 제작
5. 탈출 지점 도달 → 클리어
```
