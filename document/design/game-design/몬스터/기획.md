## 1. 컨셉 — "가짓수보다 행동 패턴"

- CORE: 몬스터의 위협은 *종 수*가 아니라 **행동/출현 패턴**(불완전 정보). "늑대가 언제 어디서" 예측이 빗나갈 여지가 핵심.
- 지역마다 **고유 위협**을 도입해 트레이드오프 압박을 갱신. 난이도 공식 `EnemyScale(d)`의 실체.
- 드롭이 **자원 흐름**([아이템 §2](../아이템/기획.md))과 직결 — 어떤 적을 잡아야 어떤 분기 자원이 나오나.

## 2. 지역별 위협 (출시 ~12~14종 + 관문 보스)

| 지역 | 몬스터(2종) | 성격·행동 패턴 | 드롭(자원) | 불완전 정보 |
|------|-------------|----------------|-----------|-------------|
| 해변 | 갯게 / 갈매기 | 약함·온보딩 | 나무·돌 | 고정(학습) |
| 숲 | 늑대(무리) / 멧돼지 | 무리 추격 / 직선 돌진 | 가죽·뼈 | **야간 활성↑** |
| 늪지 | 독개구리 / 수렁손 | 독 / 은신 기습 | 특수약초 | **안개 시 은폐** |
| 산지·폐광 | 골렘 / 동굴박쥐 | 탱크·느림 / 공중 | 철광석·뼈 | 낙석 트리거 |
| 설원 | 혹한늑대 / 설인 | 추위 부여 / 강타 | 모피 | **눈보라 시 강화** |
| 폐허 | 망령 / **관문 보스** | 다수·소모전 / 페이즈 | 고철 | 보스 패턴 변동 |

> 지역당 2종 + 관문 보스. 가짓수는 억제, **행동 다양성**으로 위협.

## 3. 행동 패턴 (불완전 정보의 원천)

현재 단순 상태머신(Idle/Walk/Hit/Death)을 **지역별 고유 행동**으로 확장:
- **무리**(늑대): 개체 약하나 떼로 → "한 마리 보이면 더 있다"
- **돌진**(멧돼지): 예고 후 직선 → 회피 타이밍
- **은신**(수렁손·안개): 시야 밖에서 기습 → 날씨와 결합한 예측 불가
- **공중/낙석**(산지): 평면 회피로 안 됨
- **날씨 강화**(혹한늑대·눈보라): 악천후 시 위협 급증 → 날씨 베팅과 직결
- **보스 페이즈**(폐허): 단계별 패턴 변화 → 최종 관문

## 4. 스케일 구조 (난이도 공식 EnemyScale)

```
EnemyScale(d) = base_enemy × (1 + k_e · d)      # d = 지역 깊이
```
- 지역 깊이에 따라 HP·ATK 배수 증가. 단 **단순 HP 인플레가 아니라 행동 위협**이 주.
- 계수 `base_enemy·k_e`, 지역별 스폰 수, 드롭률 = **level-design 밸런스 명세**.

## 5. 드롭 → 자원 흐름

- 숲(가죽·뼈) → 장비/덫 재료 / 늪(특수약초) → 고급 포션 / 설원(모피) → **방한복(설원 필수)** / 폐허(고철) → 최종 장비.
- **드롭이 다음 지역 생존의 게이트**: 설원 모피 없이는 방한복 못 만들고, 방한복 없이는 설원 동사 → 위협-자원-진행의 순환.

| 행동 | 공통 상태머신 | 지역별 고유 패턴 |
