## 지형 경사

- **슬로프 메쉬**: Synty `Ground_Slope_*` 활용. 영역별 비중:
  - 해안 0% (평지)
  - 숲 15% (드문드문)
  - 산지 50% (빽빽)
- **언덕 메쉬**: `Hill_*` 자동 배치 — 숲 1~2개, 산지 3~4개
- **랜드마크**: `Mountain_02` (스케일 1.5x, 위치 0/0/85)를 산정상 배경에 배치
- **외곽 처리** (혼합형):
  - 남쪽: 모래 + 얕은 물 (난파 컨셉)
  - 동/서: 모래 + 작은 절벽
  - 북쪽: 절벽 + 깊은 물 + 산 배경

## NavMesh

- 전체 맵에 NavMesh 굽기
- 절벽/물은 NavMesh 제외 (이동 불가 영역)
- 몬스터는 NavMeshAgent로 영역 내 배회
- **Max Slope 60°, Step Height 0.4** — 슬로프 표면 자연 이동
- Ground / Slopes / Hills + 외곽 모래만 NavigationStatic 마킹

## 마우스 시점 (Raycast)

- `PlayerCharacter.UpdateMouseLook` — `Plane(Vector3.up)` 가정 폐기
- `Physics.Raycast` + `m_GroundLayerMask`(레이어 8 "Ground")로 실제 지형 hit
- 평지/경사면 모두 마우스 위치 정확
