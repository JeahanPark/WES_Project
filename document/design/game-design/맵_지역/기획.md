# 맵 · 지역 설계 — 출시 6지역 관통

> **버전**: v1.0 (출시 개정 — 구 M2W2_05 임시 3영역 → 출시 6지역 종단)
> **작성**: 2026-06-08 (director)
> **상위**: [출시_9시간_관통구조](../출시_9시간_관통구조/기획.md) · [날씨_시스템](../날씨_시스템/기획.md)
> **성격**: director 관점(공간 구조·관문·플레이어 유도·날씨 분포)까지. 몬스터/드롭/배치 *수치*는 level-design 명세, 지형 에셋은 designer, 맵 생성 코드는 client.

## 1. 컨셉 (출시)

- **선형 종단 지형.** 난파 해변(시작) → 안전한 마을(탈출)을 **한 방향으로 길게** 관통. 9시간 단일 세션.
- **난이도 = 공간 깊이.** 마을에 가까울수록 환경이 혹독·자원이 희소.
- 각 지역은 **~1.5h 체류를 지탱하는 독립 규모** + 지역 경계 = **관문**(이전 지역의 자원·테크 없이는 못 넘음).
- **현재 임시(원형 단일섬 150u·3영역)에서 6지역 종단으로 대폭 확장** (§7 구현 영향).

## 2. 출시 지역 레이아웃 (6지역 + 마을)

> 출시_9시간 골격의 6지역을 *공간*으로 구체화. 기존 3영역(해안/숲/산지)은 초반 3지역으로 흡수·확장.

|---|------|-----------|-----------------|----------------|---------|---------|
| 0 | **난파 해변** | 평탄 모래·표류물·얕은 물 | (시작) | 절벽 위 꺼진 등대 | 맑음·흐림 | 기본(나무·돌) 풍부 |

## 3. 지역별 상세 (director 관점)

### 0. 난파 해변 — 트레이드오프 입문
- **공간**: 넓고 평탄, 위협 최소. 온보딩 구간.
- **유도**: 멀리 보이는 **꺼진 등대**(weenie)가 "저쪽으로" 끌어당김 → 자연스레 숲 방향.
- **관문 없음**(자유 학습). 첫 모닥불·기본 도구 셋업이 사실상 진행 게이트.

### 1. 숲 — 효율 vs 안전 첫 분기
- **공간**: 나무 밀집으로 시야 제한, 좁은 길이 늑대 조우를 강제.
- **유도**: **거대 고목** 랜드마크. 길이 협곡(숲길)으로 수렴 → 관문.
- **관문**: 숲길 협곡 — 야간엔 위험 급증(시간 압박 도입).

### 2. 늪지/강 — 능동 안전 + 날씨 본격
- **공간**: 수렁(이동 둔화)·강(차단). **denial space**로 목표를 가렸다 드러냄.
- **유도**: **부서진 다리**가 "건너야 한다"를 시각적으로 제시.
- **관문**: 강 건너기 — 건널 수단(테크/우회) 필요. 안개로 예측 난이도↑.

### 3. 산지/폐광 — 효율·테크 본격
- **공간**: 수직성(절벽·갱도). 고급 자원 풍부하나 낙석·골렘 고위험.
- **유도**: **산정상 윤곽**이 원경으로 계속 보여 진행 방향 고정.
- **관문**: 폐광 통과(붕괴 위험 구간) — 도구 2단계·철 장비 전제.

### 4. 설원 — 이동 비용 극대
- **공간**: 탁 트인 눈벌판(은폐 없음)·보급 희박. 체온이 핵심 자원.
- **유도**: **얼어붙은 봉우리**. 단 눈보라 시 시야 소실 → 길 잃음 압박.
- **관문**: 눈보라 횡단 — 방한 테크 없이 진입 시 빠른 동사.

### 5. 폐허 — 비가역 최종 베팅
- **공간**: 무너진 구조물 미로 + 관문 보스. 자원 최소.
- **유도**: **마을 성벽 실루엣**이 "거의 다 왔다"를 보여줌(종착 직전 긴장).
- **관문**: 마을 진입로 — 남은 자원을 탈출 푸시 vs 안전 보강에 거는 마지막 결정.

## 4. 플레이어 경로 (9시간 종단)

```
난파 해변(셋업) → 숲(효율/안전) → 늪지·강(날씨/능동안전) → 산지·폐광(효율/테크)
   → 설원(이동비용 극대) → 폐허(최종 베팅) → 마을(EscapePoint = 탈출)
```
일방향 큰 줄기 + 지역 내부 탐험 분기(보상·우회로)로 단조로움 완화.

## 5. 플레이어 유도 설계 (참고: level-design 원칙)

- **랜드마크(weenie)**: 각 지역에 원경 랜드마크 1개(등대→고목→다리→산정상→봉우리→성벽)로 다음 목표를 *명시 힌트 없이* 제시. "플레이어는 위를 안 본다" → 지평선 실루엣으로 유도.
- **denial space**: 늪지·폐허에서 목표를 일시적으로 가렸다 드러내 도달의 보상감 강화.
- **하드/소프트 경계**: 협곡·강·눈보라는 하드 경계(관문), 지역 내부는 소프트(탐험 자유).

## 6. 날씨 분포 (날씨_시스템 연동)

지역별 주 날씨(§2)는 난이도 공식 `weatherMul`과 연동. 후반 지역일수록 악천후 비중↑:
- 해변/숲: 맑음·흐림 위주(학습 구간) → 늪지: 비·안개 → 설원/폐허: 눈보라 빈발.
- 구체 출현 확률 테이블은 **level-design 밸런스 명세**.

## 7. 구현 영향 (의뢰)

| 항목 | 영역 | 내용 |
|---|---|---|
| **맵 스케일·형태 확장** | **client** | 현재 원형 섬 150u·3영역 → 6지역 종단(선형/대형). `MapGenerator` 구조 변경 — 비용 큼, 단계적 |
| **신규 지역 지형 에셋** | **designer** | 늪지·설원·폐허 지형/오브젝트 (Synty 추가 패키지 or 차용·백로그) |
| **몬스터·드롭·배치 수치** | **level-design** | 지역별 스폰 구성·드롭률·자원 밀도(CSV) — 별도 레벨 명세 |
| **지역 환경 서사** | **story** | 각 지역 분위기·환경 스토리텔링(등대·다리·폐허의 사연) |

> director는 공간 구조·관문·유도·날씨 분포까지. 위 4개는 각 영역으로 분기.

## 8. 현재 구현 (임시 — 보존, 출시 시 확장 대상)

- 원형 섬 지름 150u(PLAYABLE_RADIUS 70 / ISLAND_RADIUS 75), 3영역(해안 25%/숲 35%/산지 25%/외곽 15%).
- Synty PolygonGeneric 에셋, `MapGenerator.cs` 자동 생성, NavMesh(Max Slope 60°·Step 0.4), Ground 레이어(8), 마우스 Raycast 지형 hit.
- 슬로프/언덕 자동 배치, Mountain_02 랜드마크 손배치.
- → 출시본은 이 기반을 **6지역 종단으로 확장**(폐기 아님, 늪지/설원/폐허 신규 + 스케일업).
