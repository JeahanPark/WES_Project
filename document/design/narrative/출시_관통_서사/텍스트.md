## 1. 집필 대상·맥락

6지역 관통 단일 세션. 환경 스토리텔링 우선, 대사 최소. "먼저 온 자들"의 흔적이 모든 텍스트의 바닥 정서.

## 2. 지역 진입 내레이션 (지역 첫 진입 시 1회, 토스트/페이드)

> 화자: 무성(無聲) 내레이션. 짧고 건조하게. 진입 시 1회만.

| 지역 | 텍스트 |
|------|--------|
| 난파 해변 | `검은 파도가 너를 모래 위에 뱉어냈다.` / `절벽 위 등대는, 불 꺼진 지 오래다.` |
| 숲 | `나무들이 길을 삼킨다.` / `긁힌 자국과 흩어진 뼈 — 주인은 따로 있다.` |
| 늪지/강 | `발이 빠진다. 안개가 무엇을 숨기는지 모른다.` / `부서진 다리. 누군가 건너려 했었다.` |
| 산지/폐광 | `버려진 갱도가 입을 벌린다.` / `광부들의 도구는 그대로인데, 그들은 없다.` |
| 설원 | `눈보라가 시야를 지운다.` / `눈 속의 형체들 — 여기서 멈춘 자들이다.` |
| 폐허 | `무너진 성벽 너머, 희미한 불빛.` / `거의 다 왔다. 그런데 왜 아무도 마중 나오지 않는가.` |

## 3. 자원·아이템 플레이버 (ItemInfo.Description)

> 분위기 + 기계적 암시 균형. 건조하게.

| 아이템 | Description |
|--------|-------------|
| 나무 | `젖은 장작. 불이 붙으려면 손이 간다.` |
| 돌 | `차가운 돌. 무엇이든 되기 전의 것.` |
| 약초 | `쓴 잎. 상처에도, 추위에도.` |
| 가죽 | `질긴 짐승 가죽. 몸을 지킬 수 있다.` |
| 뼈 | `단단한 뼈. 날을 세우면 무기가 된다.` |
| 철광석 | `붉게 슨 광석. 불이 있어야 쓸모가 생긴다.` |
| 특수약초 | `늪의 독한 잎. 잘 쓰면 약, 잘못 쓰면 독.` |
| 설원 모피 | `두꺼운 짐승의 털. 이것 없이는 설원을 건너지 못한다.` |
| 고철 | `녹슨 쇳조각. 누군가의 무기였을지도.` |
| 모닥불 | `작은 불. 어둠과 추위를 잠시 밀어낸다.` |
| 철검 | `차게 벼린 칼. 손이 시리도록 무겁다.` |
| 방한복 | `누군가 남긴 외투. 그는 더 가지 못했다.` |
| 덫 | `이를 드러낸 쇠. 오지 않으면 헛것, 오면 한 번의 기회.` |

## 4. 도면 획득 통지 (즉시 토스트, 3초)

> 건조한 1줄. 느낌표 금지. "버려진 잔해에서 건진 지식".

- `{도면명} 도면을 익혔다.`
- (이미 익힘) `이미 아는 도면이다.`
- 예시 치환: "철검 도면을 익혔다." / "방한복 도면을 익혔다."

## 5. 관문·이벤트 대사 (선택적, 절제)

| 상황 | 텍스트 |
|------|--------|
| 첫 모닥불 점화 | `불이 붙는다. 잠시뿐이라는 걸 안다.` |
| 강 도달(건널 수단 없음) | `물이 깊다. 이대로는 건널 수 없다.` |
| 눈보라 진입(방한 없음) | `살을 에는 바람. 몸이 빠르게 식는다.` |
| 관문 보스 등장 | `무언가 길을 막아선다. 마지막 문턱이다.` |
| 동료 사망(멀티) | `{player_name}이(가) 쓰러졌다.` |
| 동료 부활(멀티) | `{player_name}이(가) 다시 일어선다.` |

## 6. 엔딩 / 게임오버

| 결말 | 텍스트 |
|------|--------|
| 탈출 성공 | `마을의 문이 열린다.` / `너는 살아남았다 — 이번엔.` |
| 탈출(다회차 불안) | `불빛 아래, 낯선 눈들이 너를 본다. 이곳은 정말 안전한가.` (※ director 결정 시) |
| 전멸(추위) | `추위가 마지막 숨을 가져갔다.` / `다음 난파자가 너의 흔적을 보게 될 것이다.` |
| 전멸(전투) | `더는 일어설 수 없다.` / `섬은 또 하나의 이름을 삼켰다.` |

## 7. placeholder 변수표

| 변수 | 의미 | 예시 |
|------|------|------|
| `{player_name}` | 플레이어 이름 | "생존자 2" |
| `{도면명}` | 해금된 도면 이름 | "철검" / "방한복" |
| `{지역명}` | 현재 지역 | "설원" |

## 8. 반영 위치 (client 인터페이스)

| 텍스트 | 반영 대상 | 비고 |
|--------|-----------|------|
| 지역 진입 내레이션 | 신규 UI(지역 진입 토스트) | client 신규 위젯 |
| 자원·아이템 플레이버 | `ItemInfo.csv` Description 컬럼 | 기존 컬럼, 값만 |
| 도면 통지 | 도면 해금 토스트(core-tension-fx G-9) | 기존 연출 |
| 관문·이벤트 대사 | 이벤트 트리거 UI | client 신규 |
| 엔딩/게임오버 | 클리어/게임오버 화면 | 기존 화면 문구 교체 |

> Name(이름) 컬럼은 별도 — 몬스터/아이템 이름도 story 집필 영역(후속). 본 문서는 플레이버·내레이션·문구 중심.
