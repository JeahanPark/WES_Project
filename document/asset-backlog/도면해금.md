## 신규 백로그 항목

### B-1. 도면 월드 메쉬 (두루마리/양피지 3D) — 우선순위: 낮

| 항목 | 내용 |
|---|---|
| 자산명 | 도면(두루마리) 월드 드롭 메쉬 1종 |
| 용도 | 도면 아이템이 월드에 배치/드롭될 때 보이는 3D 오브젝트. 줍기 전까지 월드에 존재(기획 §5.2) |
| 우선순위 | **낮** — 임시로 기존 `TestItem` 프리팹 메쉬 재사용 가능(코드명세 부록 A 주석). 스폰·줍기 기능엔 영향 없음 |
| 시각 요구 | 낡고 바랜 양피지 두루마리. "버려진 잔해에서 건진 지식" 톤(기획 §7). 어둡고 건조. 화사한 색 금지 |
| 임시 처리 | `ItemInfo.csv`의 도면 3종 PrefabKey에 기존 `TestItem` 재사용(스폰만 되면 됨) |
| 대체 출처 후보 | Synty PolygonPrototype/Generic props 내 두루마리/종이류 검색 → 없으면 Quaternius 무료 props 팩, 또는 AI 메쉬 생성 |
| 비고 | 전용 양피지 월드 메쉬는 "후속 권고"(코드명세 부록 A). 본 슬라이스 범위 밖 |

## 해결 완료 (procedural 3단계로 자체 처리 — 외부 자산 불필요)

| 자산 | 처리 단계 | 결과 |
|---|---|---|
| 도면 아이콘 ×3 (방패/철검/갑옷) | 3단계 Procedural | ✅ `blueprint_shield_icon` / `blueprint_ironsword_icon` / `blueprint_leatherarmor_icon` 양피지+잉크 라인 설계도 톤으로 코드 생성 |
| 잠금 오버레이 sprite | 3단계 Procedural | ✅ `lock_overlay` (반투명 어두운 베일 + 무채색 자물쇠) |
| 획득 토스트 프리팹 | 3단계 (generate_ui + 보정) | ✅ `BlueprintToast.prefab` (CanvasGroup + Background + MessageText, NotoSansKR, 양피지색 1줄) |
| (선택) 해금 강조 프레임 | 3단계 Procedural | ✅ `blueprint_highlight_frame` (바랜 양피지 테두리) |

> 아이콘 톤이 디렉터 의도(바랜 양피지·찢긴 설계도, 화사한 골드/네온 금지)에 부합하는지 최종 판단은 디렉터·QA 검토. procedural 결과물이라 미세 조정 여지 있음 — 의도 불일치 시 `BlueprintUnlockSetup.cs` 파라미터(팔레트/심볼) 수정으로 재생성 가능.
