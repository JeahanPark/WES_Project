### 🚨 자동 플래그 (코드 검사)

자동 플래그 도구가 제한적이라 결정론적 검출이 부분적임. 다음 사항은 코드 분석으로 확정:

1. **[A4] CraftPopup `m_HintText` 참조 누락 의심**
   - [CraftPopup.cs:11](Assets/Scripts/UI/Popup/CraftPopup/CraftPopup.cs#L11)에서 `[SerializeField] m_HintText` 선언, [55-63](Assets/Scripts/UI/Popup/CraftPopup/CraftPopup.cs#L55) `SetHintVisible`이 null 가드로 묵묵히 패스
   - 화면(스크린샷 02·03)의 디테일 패널 영역에 안내 텍스트가 안 보임 → 프리팹 미연결 추정
   - 화면: `screenshots/2026-04-28-0904/02_CraftPopup_building.png`

2. **[A4] InventoryPopup `OnInventoryChanged` 미구독**
   - [InventoryPopup.cs](Assets/Scripts/UI/Popup/InventoryPopup/InventoryPopup.cs)의 `Start()`에서 `RefreshInventory()` 1회만 호출, `inventory.OnInventoryChanged` 이벤트 구독 없음
   - 팝업이 열린 상태에서 아이템 추가/제거 시 그리드 미갱신 — QA 중 SimulateAddItems(5,10) 호출 후 그리드에 반영 안 됨 확인
   - 화면: `screenshots/2026-04-28-0904/04_InventoryPopup_with_items.png`

### ⚠️ 시각 검수: 문제 발견

#### CraftHUDTab (HUD 좌측 사이드 메뉴)

3. **[B2] 버튼 크기 과소 — 클릭 영역 좁음**
   - "건축" / "인벤토리" 버튼이 화면 좌측 가장자리에 매우 작게 배치됨
   - "인벤토리" 라벨이 버튼 폭을 거의 꽉 채워 클리핑 직전
   - 화면: `screenshots/2026-04-28-0904/01_CraftHUDTab_default.png`

4. **[B4] HUD 좌측 패널 시각 분리 약함**
   - `PlayerStatusHUD`(HP/Cold)와 `CraftHUDTab`(건축/인벤토리) 모두 좌측에 배치되어 있으나 그룹 구분(여백/배경)이 약함
   - 같은 좌측 컬럼에 다른 위계의 UI가 한꺼번에 들어가 있어 시각적 정보 분류가 어려움

#### CraftPopup

5. **[B5] DetailPanel 빈 상태 가이드 없음**
   - 카테고리 진입 직후 디테일 패널은 비활성(`Hide()`) 상태여야 하나, 화면(02·03)에서 판단할 때 패널 영역에 안내가 보이지 않음 (HintText 미배치)
   - "재료를 선택하세요" 같은 안내 텍스트가 필요

6. **[B2] 좌·우 비율 불균형**
   - 좌측 `LeftPanel`(셀 스크롤)이 화면의 1/4 정도, 우측 디테일 패널이 1/2 이상 차지
   - 셀 아이콘이 매우 작아 항목 식별성 떨어짐
   - 좌측 영역 대부분이 빈 공간 — 셀 폭 확대 또는 그리드 레이아웃 권장

7. **[B3] 팝업 뒤 게임 월드 비침**
   - Dim 오버레이가 없어 팝업 외 영역의 게임 월드/HUD가 또렷이 보임
   - 사용자 주의 분산, 클릭 영역 모호
   - 화면: `screenshots/2026-04-28-0904/02_CraftPopup_building.png`

8. **[B1] 셀 간 정렬/간격 미적용**
   - 셀 2개(모닥불·횃불)가 좌상단에 가깝게 붙어 있고, 셀 사이 간격이 작음
   - VerticalLayoutGroup의 spacing 또는 cell 크기 조정 필요

9. **[B6] 셀 hover/disabled 상태 시각 미확인**
   - 코드상 `SetSelected`로 Selected 시각화 추가됨 (이번 작업)
   - 하지만 hover, disabled(재료 부족 시 셀 회색 처리 등) 상태 처리 없음

#### InventoryPopup

10. **[B5] 빈 슬롯 그리드 미표시 (P10 재확인)**
    - 슬롯 경계/배경이 보이지 않음 → 인벤토리 용량 파악 어려움
    - `m_BackgroundImage` 필드는 존재하지만 프리팹에서 sprite/alpha가 비어있을 가능성
    - 화면: `screenshots/2026-04-28-0904/04_InventoryPopup_with_items.png`

11. **[B2] DetailPanel 영역 미식별**
    - 우측 패널이 시각적으로 보이지 않음 — 인벤토리 그리드와 시각 분리 부재
    - 사용자가 "어디에 정보가 표시되는지" 인지 불가

12. **[B3] 팝업 뒤 게임 월드 비침**
    - CraftPopup과 동일한 문제 (Dim 오버레이 없음)

13. **[B7] 인벤토리 갱신 누락 (자동 플래그 #2와 연계)**
    - 팝업이 열린 상태에서 아이템 변동이 반영되지 않음 → 사용자가 새로고침을 위해 팝업 재오픈 필요

#### 배치 모드 (BuildingPlacementWorker)

14. **[B6] MCP 환경에서 배치 시각 검증 불가**
    - 마우스 위치 입력이 (0,0)으로 고정되어 커서 아이콘/범위 인디케이터가 화면 밖에 위치
    - 코드상 P5(빨강/초록 색상 검증), 커서 아이콘은 정상 — 실 마우스 입력 환경에서 사용자 직접 확인 필요
