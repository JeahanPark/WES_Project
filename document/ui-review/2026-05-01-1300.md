## 🚨 자동 플래그 (코드/Prefab 검사)

### 1. [A4] CraftDetailPanel — Inspector 참조가 잘못된 GameObject를 가리킴 (HIGH)

[CraftPopup.prefab:1960-1961](Assets/GameResource/UI/Popup/CraftPopup.prefab#L1960-L1961) 의 `CraftDetailPanel` MonoBehaviour 필드:

| 필드 | 현재 fileID | 가리키는 GameObject | 올바른 GameObject |
|---|---|---|---|
| `m_IconImage` | 100067 | **CellTemplate/IconImage** (좌측 셀 템플릿) | DetailPanel/IconImage (fileID 100079) |
| `m_NameText` | 100071 | **CellTemplate/NameText** (좌측 셀 템플릿) | DetailPanel/NameText (fileID 100083) |
| `m_DescriptionText` | 100087 | DetailPanel/DescriptionText | ✅ 올바름 |
| `m_MaterialItemTemplate` | 100099 | DetailPanel/MaterialItemTemplate | ✅ 올바름 |
| `m_CraftButton` | 100116 | DetailPanel/CraftButton | ✅ 올바름 |

**증상**:
- 우측 디테일 패널 상단에 **"Item Name"** 이라는 placeholder가 그대로 노출됨 (실제 데이터 "모닥불"로 갱신 안 됨)
- 디테일 패널 상단 **아이콘 영역이 비어 있음** (모닥불 아이콘이 셀에는 보이는데 디테일에는 안 뜸)
- 이유: [CraftDetailPanel.Show()](Assets/Scripts/UI/Popup/CraftPopup/CraftDetailPanel.cs#L36-L48) 에서 `m_NameText.text = _craftInfo.Name` 와 `m_IconImage.sprite = ...` 를 호출해도, 실제로는 **숨겨진 CellTemplate** 의 텍스트/이미지를 갱신하고 있어서 화면상 변화가 없음.

## ⚠️ 시각 검수: 문제 발견

### 2. [B1+B4] 디테일 패널 본문 텍스트 가독성 (MID)
- 설명 텍스트("체온을 올려주는 모닥불입니다")가 **너무 작고 폰트 색이 어두워** 거의 안 보임
- "필요 아이템", "제작 조건" 라벨은 보이지만, 본문 영역과 좌우 여백이 과해 시선이 분산됨

### 3. [B4] 재료 행(Material Row) 스타일 불일치 (MID)
- 동적으로 Instantiate 되는 재료 행이 **밝은 청록/녹색 배경의 큰 박스**로 노출됨
- 팝업 전체 톤(어두운 회갈색)과 명백히 안 맞음 — `MaterialItemTemplate` 자체의 디자인이 재사용 가능한 톤으로 정리돼 있지 않음
- 글자도 박스에 비해 작아 비례가 깨짐

### 4. [B6] 셀 선택 피드백 부재 (MID)
- 좌측 모닥불 셀을 클릭해도 **선택 상태 표시(주황 프레임)가 안 보임**
- [CraftScrollCell.cs:13](Assets/Scripts/UI/Popup/CraftPopup/CraftScrollCell.cs#L13) `m_SelectedFrame` 이 정의돼 있지만 [CraftPopup.prefab:1605](Assets/GameResource/UI/Popup/CraftPopup.prefab#L1605) 에서 `m_SelectedFrame: {fileID: 0}` (= 미연결)
- `m_BackgroundImage` 도 fileID 0 으로 미연결 → 색상 변경 피드백도 동작 안 함

### 5. [B2] 디테일 패널 내부 패딩 부족 (LOW)
- 우측 패널 내부 콘텐츠가 좌측 분리선과 너무 붙어 있고, 우측 끝에도 여백 거의 없음
- 4:6 비율 분할은 의도된 것 같으나, 분할선이 시각적으로 표현되어 있지 않아 두 영역의 경계가 모호함

### 6. [B4] 카테고리 탭(건물/아이템) 시각 처리 (LOW)
- 활성/비활성 탭 구분이 명확하지 않음 (둘 다 비슷한 회색 배경)
- 폰트 크기/색이 셀 안 라벨과 맞지 않음

### 7. [B2] CraftButton 위치/노출 (LOW)
- 우측 패널 어디에 있는지 시각적으로 파악 어려움 (스크린샷에서 보이지 않음 — Tier가 가장 아래라 디테일 영역 밖으로 밀린 것으로 추정)
- 화면 캡처 영역 안에 들어와야 사용자 액션이 자연스러움
